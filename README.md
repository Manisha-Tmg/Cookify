# Cookify

Cookify is a full-stack recipe sharing web application where users can browse, create, and review recipes. The platform allows users to add ingredients, cooking steps, images, and ratings, making it easy to discover and share delicious meals. Cookify provides an interactive experience for food lovers to explore new dishes and contribute their own recipes.

## Features
- Add and share recipes
- View detailed ingredients and step-by-step instructions
- Rate and review recipes
- User authentication and authorization
- Upload images for recipes
- Responsive and user-friendly interface
